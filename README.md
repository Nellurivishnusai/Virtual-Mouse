This is a project built keeping in mind the motor and physical limitation in certain or permanent situation.
I wanted to make a basic hand tracking mouse model which could be upscaled later.
I integrated AI bots such as ChatGPT, Claude and Gemini as an assist to this porject.
I used these AI bots to gain knowledge about the technologies I needed to be aware of and have knowledge of to pull this off.
After knowing about what I was working with, my next question was How and Where do I start.
I then took help of ChatGPT to break down my problem statement into clear theoretical parts, so that I could focus on building it part by part.
Then, I used Claude and Gemini to raw code each part while testing it and making changes in the code as per my requirement.
I wanted the tool I was creating to be user friendly, so I took Chat GPT's suggestion of displaying the gestures my projet provides.
I started from scratch, more of my work progress can be reflected in my repository and it's changes.
